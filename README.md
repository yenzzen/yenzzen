### Projetos

#### [Kwo](https://github.com/yenzzen/kwo-showcase) — Governança de Código via IA
Plataforma que permite a Tech Leads definirem regras de arquitetura em **linguagem natural**, aplicando-as em tempo real diretamente no IDE dos desenvolvedores.
* **Destaque:** Motor de inferência híbrido com processamento paralelo e semáforos para análise semântica contextual.
* **Tech Stack:** Go 1.25, React 19, TypeScript, SQLite, LLM Integration.
<br>

#### [LazyReact](https://github.com/yenzzen/lazyreact-showcase) — Framework Backend-Agnostic
Framework que desacopla o React do ecossistema Node.js, permitindo que qualquer backend (Go/Rust/Python) hidrate aplicações nativamente via **Props Injection**.
* **Destaque:** Binário único de deploy (~20MB) com build do frontend em menos de 100ms e zero loading states.
* **Tech Stack:** Go 1.25, React 19, esbuild, ESM + Import Maps.
<br>

#### [ApodBrazil](https://www.apodbrazil.com/) — Curadoria Astronômica
Plataforma full-stack em produção há mais de um ano, gerenciando o acervo da astrofotografia brasileira.
* **Destaque:** Sistema de votação para jurados e visualizador interativo com zoom matemático de alta precisão.
* **Tech Stack:** Next.js 16, Supabase, AWS S3, TailwindCSS.
